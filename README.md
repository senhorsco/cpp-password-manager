# 🔐 C++ Encrypted Password Manager

A lightweight, console-based password management application developed in C++. This project demonstrates core computer science principles, including Object-Oriented Programming (OOP), File I/O operations, and fundamental cryptography using a custom XOR algorithm.

## ✨ Features
* **Custom Encryption:** Utilizes an XOR bitwise operation to encrypt passwords before they are written to the disk.
* **Secure Storage:** Writes only ciphertext to local `.dat` files, ensuring data is unreadable if the file is manually opened.
* **Data Encapsulation:** Built with OOP principles to keep data processing secure and modular.
* **No External Dependencies:** Built entirely with standard C++ libraries (`<iostream>`, `<fstream>`, `<string>`).

## 🛠️ Technologies Used
* **Language:** C++
* **Concepts:** OOP, File I/O, Bitwise Operations, Cryptography
